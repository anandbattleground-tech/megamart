## Design Assessment

### What to Preserve (Non-Negotiable)
- ✅ Logo — exactly as-is (colorful star, red banner, ChunniLal script, tagline)
- ✅ All real photography (store exteriors, category models, furnishing images)
- ✅ All brand partner logos
- ✅ All store addresses and phone numbers
- ✅ Social media platform links (FB, IG, YT, WA)
- ✅ "Something for Everyone" category structure (Men, Women, Kids)
- ✅ Home Furnishing as a product category
- ✅ Stats: Satisfied Customers | All Major Brands | 5 Stores in Bihar

### What is Generic / Needs Improvement
- ❌ **No hero headline or CTA on home page** — fabric image has no message or call-to-action
- ❌ **Flat, dated layout** — no visual hierarchy beyond basic stacking
- ❌ **No animations** — completely static, no scroll reveals, no transitions
- ❌ **Broken images on Contact page** — Raja Bazar and Biharsharif cards have no images
- ❌ **Weak typography** — no visual contrast between display/body text
- ❌ **No mobile-optimized feel** — layout appears desktop-only
- ❌ **No stats with numbers** — "Satisfied Customers" has no figure (e.g., "10,000+")
- ❌ **No testimonials section** — missing social proof entirely
- ❌ **Welcome bar text effect** — typewriter animation present but basic
- ❌ **Brand carousel pagination** — uses basic dots, not interactive/smooth
- ❌ **About page is text-heavy** — needs visual breakup, icons, or timeline
- ❌ **Footer is cramped** — all store addresses in one text block, hard to scan

### Opportunities in Redesign
- Add a strong hero headline + CTA button ("Visit Us Today" / "Explore Collections")
- Add an animated stats counter (e.g., "10,000+ Happy Customers", "5 Stores")
- Add scroll-triggered fade-in/slide animations throughout
- Upgrade brand carousel to smooth auto-scroll
- Create a visual store locator card grid (replace broken-image placeholders)
- Add a testimonials section
- Improve About Us with visual elements (icons for values, timeline, or image grid)
- Improve mobile responsiveness
